# AGENTS.md

## Customization Rules

- 生成的实施计划（implementation_plan.md）、任务列表（task.md）以及任何 Walkthrough 文档（walkthrough.md）都必须使用中文编写。
- The generated implementation plan (implementation_plan.md), task list (task.md), and walkthrough.md must be written in Chinese.
- 每次增加大功能、修改核心功能、删除功能时，必须将其记录更新到 `CHANGELOG.md` 中。
- Every time you add major features, modify core functions, or delete functions, you must update the changes to `CHANGELOG.md`.
